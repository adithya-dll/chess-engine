# Chess Engine

This is a simple chess engine with a graphical user interface built using Pygame. You can play against a basic AI that uses the minimax algorithm to determine its moves.

## How to Run

1.  **Install Pygame:**
    ```bash
    pip install pygame
    ```

2.  **Run the game:**
    ```bash
    python main.py
    ```

## Features

*   **Playable Chess Game:** A fully playable chess game with a graphical interface.
*   **Minimax AI:** The opponent is controlled by a simple AI that uses the minimax algorithm with alpha-beta pruning to find the best move.
*   **Castling:** The game correctly implements castling.
*   **Checkmate and Stalemate:** The game detects checkmate and stalemate conditions and ends the game accordingly.
*   **Undo Move:** You can undo the last move by pressing the 'z' key.
